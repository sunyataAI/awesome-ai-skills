# Contributing

Thanks for helping improve Awesome AI Skills.

感谢你帮助完善 Awesome AI Skills。

## What Makes a Good Entry / 什么项目适合收录

A recommended project should have at least one of these qualities:

推荐项目最好至少满足以下一项：

- It provides a reusable AI skill, workflow, agent pattern, or integration.
- It helps developers understand how to build AI-native software.
- It has clear documentation or readable implementation.
- It is actively maintained or historically important.
- It offers a practical pattern that SunyataAI builders and readers can learn from.

## Required Information / 必填信息

Please include:

请提供：

- Project name / 项目名称
- Repository URL / 仓库链接
- Category / 分类
- Short Chinese or English summary / 简短中文或英文介绍
- Why it is worth tracking / 为什么值得关注

## Style Guide / 写作风格

- Be factual and concise.
- Avoid promotional language.
- Prefer specific use cases and concrete architecture notes.
- Mark uncertainty explicitly when information is not verified.
- Write bilingual entries when possible.
